# My Work — Aarati (installable Android version)

This package is a real Progressive Web App (PWA) with a web manifest, icons and a service worker. Those files are required for Chrome to offer installation.

## Upload all five files to GitHub Pages
Upload these files into the root of a new **public** GitHub repository:

- `index.html`
- `manifest.webmanifest`
- `service-worker.js`
- `icon-192.png`
- `icon-512.png`

Do **not** upload the ZIP file itself to GitHub; upload the files inside it.

Enable GitHub Pages:
`Settings → Pages → Deploy from a branch → main → /(root) → Save`

## Install on Android
1. Open the published `https://USERNAME.github.io/REPOSITORY/` link in **Chrome**.
2. Refresh once, then wait a few seconds.
3. Tap the in-app **Install My Work** button. If Chrome has not shown its prompt, tap Chrome's **⋮** menu and choose **Install app** or **Add to Home screen**.
4. Confirm the name **My Work**.

The app stores Aarati's shifts locally in Chrome on that Android phone. Use **Export spreadsheet (CSV)** as a backup.
